Run TopPIC Suite using Docker

1 Install docker

Go to the Docker website, choose the OS from the left, and follow the instructions to install docker engine.

2 Pull image from the repository

Enter the command below to pull the docker container image from the TopPIC repository.

docker pull 'repository name'/'image name'

3 Run TopPIC

If the data files are in the folder /home/usr/Documents/Toppic_data, run the command as following. The outputs are also going to be stored in the same folder.

docker run -it -v /home/usr/Toppic_data:/data 'repo_name'/image name' /bin/bash

Now TopPIC suite can be run using commands following the format below: 

../opt/toppic/bin/'progam_name' 'datafile_name'

For example, to run TopFD, 

../opt/toppic/bin/topfd st_*.mzML

For Toppic, 

../opt/toppic/bin/toppic -f C57 -d -t FDR -T FDR -c combined uniprot-st.fasta st_*_ms2.msalign

For TopMG, 

../opt/toppic/bin/topmg -f C57 -d -t FDR -v 0.05 -T FDR -V 0.05 -i variable_mods.txt uniprot-st.fasta st_1_ms2.msalign

For TopDiff, 

../opt/toppic/bin/topdiff -f C57 uniprot-st.fasta st_1_ms2.msalign st_2_ms2.msalign
