TopPIC Gateway

TopPIC Gateway (https://toppic.sciencegateways.iu.edu/) provides users with a web-based system to utilize tools in TopPIC Suite to analyze top-down mass spectrometry data using high-performance computing resources, like Jetstream and Indiana University Big Red 3. The platform uses the Apache Airavata science gateway middleware to manage and schedule tasks submitted by users. More details about TopPIC Gateway can be found in the TopPIC Gateway paper.

To log into the TopPIC Gateway, the user needs to do the following steps.
1. Click the "Log in" button at the top right corner of the web page of TopPIC Gateway.
2. Select "Sign in with Existing Institution Credentials."
3. Use your University ID to log in. If your university is not included in the list of "identity providers," you can use "Google" as your identify provider and your Gmail account as your ID.
